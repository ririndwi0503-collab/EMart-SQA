# Software Test Plan (STP)

## 1. Pendahuluan
Dokumen ini berisi rencana pengujian sistem EMart untuk memastikan 
sistem berjalan sesuai kebutuhan.

## 2. Tujuan Pengujian
- Memastikan login berjalan
- Memastikan register berjalan
- Memastikan produk tampil
- Memastikan cart berjalan
- Memastikan checkout berjalan

## 3. Lingkungan Pengujian
- Browser Chrome
- Windows 10
- Internet connection

## 4. Test Case

### Test Login
Input: username dan password
Output: berhasil login

### Test Register
Input: data user
Output: akun berhasil dibuat

### Test Add Cart
Input: klik add to cart
Output: produk masuk cart

### Test Checkout
Input: klik checkout
Output: transaksi berhasil

## 5. Kriteria Keberhasilan
Sistem dinyatakan berhasil jika semua fitur berjalan tanpa error.
